## 2. Tareas en las que se Usó la IA

- **Búsqueda y entendimiento de funciones nuevas:** por ejemplo la función `detect()` de la librería `langdetect` para identificar el idioma de cada reseña antes de filtrar el corpus.
- **Estructurar mejor el codigo:** se uso la IA para poder organizar mejor el codigo y una mejor forma de verlo
- **entendimiento del proyecto:** se usó la IA para explicar el documento pdf de otra manera para tener mas conocimiento sobre lo que estamos trabajando
